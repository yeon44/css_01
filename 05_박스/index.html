Padding 작성시, padding: 20px; 상우하좌의 padding 값이 동일한 경우 스타일 적용시 사용

Padding 작성시, padding-top: 30px; padding-right: 20px; padding-bottom: 40px; padding-left: 10px; 으로 각각 다른 값으로 스타일을 부여하고 싶다면, 상단으로부터 시계방향으로 값을 넣음.
padding: 30px 20px 40px 10px / padding: 상 우 하 좌

padding: 30px 20px 40px 10px

Padding 작성시, padding-top: 20px; padding-right: 40px; padding-bottom: 20px; padding-left: 40px; 으로 스타일을 부여하고 싶다면 통합시 상단으로부터 시계방향으로 각각 방위에 대한 부여가 가능하되 통합도 가능하다.
padding: 20px 40px 20px 40px; padding: 상 우 하 좌;
padding: 20px 40px; padding: (상하) (우좌)

padding: 20px 40px

Padding 작성시, padding-top: 60px; padding-right: 40px; padding-bottom: 20px; padding-left: 40px; 스타일을 부여하고 싶다면 통합시 상단으로부터 시계방향으로 각각 방위에 대한 부여가 가능하되 통합도 가능하다.
padding: 60px 40px 20px 40px; padding: 상 우 하 좌;
padding: 60px 40px 20px; padding: 상 (우좌) 하;

padding: 60px 40px 20px
