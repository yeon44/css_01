배경색상(background-color) : 문자(red, blue, ...) / 16진수 코드(#ff0000 -> #f00) / hsl(색상(deg), 채도(%), 명도(%)) / rgba(red(0~255), green(0~255), blue(0~255), alpha(0~1)) -> rgba(255, 0, 0, 0.5)

배경이미지(background-image:url(이미지 파일의 경로))

배경의 위치 선언(background-position: 수평위치 수직위치)

수평 위치선언 : left, center, right, 0~100%, px

수직 위치선언 : top, center, bottom, 0~100%, px

배경의 사이즈 조정(background-size : cover / contain / 가로px 세로px / 가로px auto / auto 세로px)

cover : 가로사이즈가 세로사이즈보다 긴 경우, 화면의 높이를 기준으로 맞추겠다

contain : 세로사이즈가 가로사이즈보다 긴 경우, 화면의 세로기준으로 맞추겠다. & 로고를 배경으로 보여주고자 할 때, 로고의 영역이 모두 보이도록 구성해주는 선언

배경의 반복(패턴) 설정(background-repeat)

배경의 parallax 효과(background-attach: scroll(기본값) / fixed)
